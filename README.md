# Rezept Website API

Eine einfache API für die Rezept Website zum erstellen und suchen von Rezepten.

Implementierte Funktionen:

- Rezepte schreiben, anzeigen und löschen.
- Rezensionen schreiben, anzeigen, ändern und löschen.
- Benutzerkonto erstellen, einloggen, ausloggen und löschen.
- Suchen nach Rezepten (es werden nur Rezepte angezeigt, die eines der eingegebenen Wörter beinhalten).

Verwendete Technologien:

- [Node.js](https://nodejs.org/en/) und [Express](https://expressjs.com/) für die Serverseitige Programmierung.
- [MongoDb](https://www.mongodb.com/) und [Mongoose](https://mongoosejs.com/) als Datenbank.
- [Passport](http://www.passportjs.org/) für die Authentifizierung.
